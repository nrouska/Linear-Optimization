# Optimal Diet Problem
Project developed by Natalia Rouska. Solves the "Stigler Diet Problem" using Linear (LP) and Mixed-Integer Programming (MILP) to find a daily meal plan that meets nutritional requirements at the lowest cost.

Data: USDA (64 foods and 8 nutrients)
